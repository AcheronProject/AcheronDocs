Guidelines¶

  • Grounding, bypassing and oscillator tracing: Application Note AVR186 : best practices for the PCB layout of oscillators. Although rather short, this AN contains valuable information on the guidelines and layout of crystal oscillators in MCUs. This thread , specially the response by Olin Lathrop, on the Electronics StackExchange, also contains good information on crystal oscillator tracing and grounding on the PCBs; in the same spirit, the response from the same person in this tread contains further more detailed information on the same subject, but also giving good detailed analysis on grounding and bypassing. In the same thread, Mike DeSimone also gives insightful information on these subjects.
