Manufacturing files¶

In the ./gerbers folders you can find the plot files for the PCB.

Attention

Beware that the PCBs are released under a no-warranty and no-liability license, meaning that I cannot guarantee they are fit for a specific purpose. Always check the manufacturing files before ordering their manufacture so that you can be sure they are fit for your purpose. For gerbers I recommend tracespace.io gerbers viewer.

PCB Gerber files¶

This zipped folder contains the Gerber files for the last PCB version. Among them are probably more files than needed, including the files for the PCB edges and drill files.

Attention

The ElongatePCB introduction page contains badges to show the last prototyped version of the PCB. I will release un-prototyped Gerbers in order to prototype them. Always check if the version you are downloading was prototyped!
